### Development

Run a simple HTTP web server:

```
# Ruby 1.9.2+
ruby -run -e httpd . -p 8080

# Python 3
python -m http.server 8080
```

Go to [http://localhost:8080](http://localhost:8080)

Refresh with `CTRL+F5` if the web server caches files.
